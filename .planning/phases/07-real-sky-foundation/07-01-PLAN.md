Pipeline stages (all sharp, all recipe-locked from 07-RESEARCH.md §1, verified there against the real file):
  1. DOWNLOAD + INTEGRITY GATE: if `sky-source/noirlab2430b-4k.tif` is absent, download it via curl/fetch from `https://noirlab.edu/public/media/archives/images/publicationtiff/noirlab2430b.tif`.   2. CANDIDATE CROPS: generate 2-3 candidate column windows within the verified safe row-range y=550..1450 (the galactic-core heuristic at rows ~1000-1090, cols ~1670-2470 is the primary candidate; vary the column window for the other 1-2), per 07-RESEARCH.md §1.3 Open-Question 1. For each candidate: `extract()` the safe-zone band, `rotate()` ~20-24deg (target the UI-SPEC diagonal: top x:0.63 -> horizon x:0.87, ~24deg from vertical) with `{ background: '#0f1216' }` (--bg, corner fill), then `extract()` the largest-inscribed rectangle using the exact formula in 07-RESEARCH.md §1.4 (never a hand-picked percentage — compute it so no fill leaks into the crop).
  3. GRADE (build-time bake, per 07-UI-SPEC Tone Contract): black point of non-galactic dark sky graded to RGB (5,7,10) +/-4 = --sky-zenith #05070a; near-horizon non-band ceiling not exceeding --sky-horizon #141a2c; saturation of bulk dust/sky toward the blue-slate family (hue 200-230deg, HSL S <= 15%) via sharp levels/modulate/tint — desaturate any green/red airglow anomaly in the chosen crop. COLUMN VIGNETTE (replaces the runtime governor): composite a masked darken pass toward #05070a using the IDENTICAL geometry as the SKY-05 governor — half-width 464px + 80px smoothstep ramp at the 1440-ref width, scaled proportionally to each export width — full darken inside the column (visually equivalent to the procedural 0.12x alpha reduction), ramping to zero outside.
  4. SEAM RAMP: bake the vertical alpha blend to --bg via the SVG-linearGradient composite technique (07-RESEARCH.md §1.5) with stops 0..0.72 pure photo, 0.72..0.94 ramp photo -> #0f1216, 0.94..1.0 solid #0f1216 (07-UI-SPEC Seam Contract).
  5. COLOR-PROFILE NORMALIZE: call `.toColorspace('srgb')` before encode (07-RESEARCH.md §7 pitfall 4) so the embedded ICC profile can't shift on wide-gamut displays.
  6. EXPORT for the winning candidate: resize to 2560w and 1920w; encode AVIF `{ quality: 58, bitdepth: 10, chromaSubsampling: '4:4:4', effort: 6 }` and WebP `{ quality: 72 }` -> public/sky/milky-way-{2560,1920}.{avif,webp}; write a ~32w WebP q40 LQIP base64 data string to public/sky/milky-way-lqip.txt (07-RESEARCH.md §1.7 byte budgets). Author the source crop at a wide ~2.3:1-2.4:1 letterbox so object-fit:cover never reveals past content (07-UI-SPEC Master authoring).
  7. EVIDENCE: also write the 2-3 candidate previews (JPEG) to .planning/phases/07-real-sky-foundation/spike-banding/ for the verdict + eyeball task.
